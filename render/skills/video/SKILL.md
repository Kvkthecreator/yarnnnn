# Video Generation Skill

Produces short-form video clips (max 30 seconds) from slide-based specifications via Remotion.
Each slide contains elements (text, values, lists) with layout modes and entrance animations.

## Input Spec

```json
{
  "title": "Weekly Competitive Brief",
  "orientation": "landscape",
  "theme": {
    "background": "#0f172a",
    "foreground": "#ffffff",
    "accent": "#3b82f6",
    "muted": "#94a3b8"
  },
  "slides": [
    {
      "layout": "center",
      "duration": 4,
      "transition": "fade",
      "elements": [
        {"type": "heading", "text": "Weekly Brief", "size": "2xl"},
        {"type": "text", "text": "April 3, 2026", "size": "sm", "color": "muted"}
      ]
    }
  ]
}
```

### Top-Level Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `title` | string | no | Video title (metadata only) |
| `orientation` | string | no | "landscape" (1920x1080) or "portrait" (1080x1920). Default: landscape |
| `theme` | object | no | Colors: background, foreground, accent, muted (hex values) |
| `slides` | array | yes | 1-8 slides. Total duration must not exceed 30 seconds |

### Slide Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `layout` | string | no | "center", "stack", or "split". Default: center |
| `duration` | number | yes | Seconds this slide is shown (1-15) |
| `transition` | string | no | "fade", "slide-left", "slide-up", "cut". Default: fade |
| `elements` | array | yes | Elements to render on this slide |

### Element Types

| Type | Props | Description |
|------|-------|-------------|
| `heading` | text, size (sm/md/lg/xl/2xl/3xl), color | Large text with spring entrance |
| `text` | text, size (sm/md/lg), color | Body text with fade entrance |
| `value` | text, size (xl/2xl/3xl), color | Large number with scale animation |
| `badge` | text, color (green/red/yellow/blue/accent) | Pill badge with pop animation |
| `list` | items[] | Bulleted list with stagger entrance |
| `divider` | — | Animated horizontal line |
| `spacer` | height | Vertical spacing |

### Layouts

| Layout | Description |
|--------|-------------|
| `center` | All elements vertically + horizontally centered |
| `stack` | Top-aligned, elements flow down |
| `split` | Left/right columns (elements specify `position: "left"/"right"`) |

### Colors

Elements accept `color` as: `"foreground"`, `"muted"`, `"accent"`, or a hex value.
Badge accepts: `"green"`, `"red"`, `"yellow"`, `"blue"`, `"accent"`.

## Constraints

- **Maximum 30 seconds** total duration across all slides
- **Maximum 8 slides** per video
- **Silent** — no audio support
- **1080p** — landscape (1920x1080) or portrait (1080x1920)
- **Render time** — 30-90 seconds depending on complexity
- **Format** — MP4 (H.264) only
