# Image Composition Skill

Creates composed images (social cards, banners, header images) from structured specs using Pillow.

## Input Spec

The input is a JSON object:

```json
{
  "layout": "card",
  "width": 1200,
  "height": 630,
  "background": "#1a1a2e",
  "elements": [
    {
      "type": "text",
      "content": "Main Heading",
      "x": 60,
      "y": 200,
      "font_size": 48,
      "color": "#ffffff",
      "bold": true
    },
    {
      "type": "text",
      "content": "Subtitle or description text",
      "x": 60,
      "y": 280,
      "font_size": 24,
      "color": "#a0a0b0"
    },
    {
      "type": "rect",
      "x": 0,
      "y": 0,
      "width": 8,
      "height": 630,
      "color": "#e94560"
    }
  ]
}
```

### Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `layout` | string | no | Layout preset: `card` (1200x630), `banner` (1200x300), `square` (1080x1080). Overridden by explicit width/height. |
| `width` | integer | no | Canvas width in pixels (default: from layout or 1200) |
| `height` | integer | no | Canvas height in pixels (default: from layout or 630) |
| `background` | string | no | Background color as hex (default: `#ffffff`) |
| `elements` | array | yes | Array of elements to render |

### Element Types

**text:**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `content` | string | yes | Text to render |
| `x`, `y` | integer | yes | Position (top-left origin) |
| `font_size` | integer | no | Font size in pixels (default: 24) |
| `color` | string | no | Text color as hex (default: `#000000`) |
| `bold` | boolean | no | Use bold weight (default: false) |

**rect:**
| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `x`, `y` | integer | yes | Position |
| `width`, `height` | integer | yes | Dimensions |
| `color` | string | yes | Fill color as hex |

## Output Formats

- `png` — Lossless. Best for graphics with text.
- `jpg` — Lossy (quality 90). Smaller file size for photos.

## Examples

### Social card
```json
{
  "type": "image",
  "input": {
    "layout": "card",
    "background": "#0f172a",
    "elements": [
      {"type": "rect", "x": 0, "y": 0, "width": 6, "height": 630, "color": "#3b82f6"},
      {"type": "text", "content": "Q2 Revenue Report", "x": 60, "y": 220, "font_size": 44, "color": "#ffffff", "bold": true},
      {"type": "text", "content": "Generated by YARNNN • March 2026", "x": 60, "y": 290, "font_size": 20, "color": "#94a3b8"}
    ]
  },
  "output_format": "png"
}
```

## Constraints

- Maximum canvas: 4096x4096 pixels
- Font: DejaVu Sans (bundled in Docker image)
- No image embedding from URLs (text and shapes only)
- No gradients or shadows — solid fills only
- PNG output is not optimized (no pngcrush)
