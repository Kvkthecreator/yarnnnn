# Slack Integration

Connect Slack so YARNNN can read team conversation and turn it into useful supervised output.

## What YARNNN can do with Slack

- answer questions grounded in channel history
- produce recap and digest agents
- detect recurring themes, blockers, and decision threads
- combine Slack context with Gmail, Notion, and Calendar

## Setup

1. Go to **Context** → click **Connect** on Slack
2. Authorize YARNNN in the Slack login screen
3. Let YARNNN apply smart default channel coverage for the first sync
4. Refine channel coverage later if needed

**YARNNN is read-only** — it reads messages but never posts, reacts, or modifies anything in your Slack workspace.

## Tips

- start with channels where decisions and blockers actually show up
- both public and private channels can be useful if you are a member
- narrow coverage if the first recap is too noisy
- expand coverage only after the first agent is useful

## Example agents using Slack

- weekly engineering recap
- cross-team product update
- watch agent for urgent or recurring issues
