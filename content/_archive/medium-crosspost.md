# Template: Medium Cross-Post

**Voice:** YARNNN brand
**Read before drafting:** `_voice/yarnnn-brand-voice.md`
**Source:** Always cross-posted from yarnnn.com/blog — Medium is the amplifier, not the source of truth.

---

## Process

1. **Import from blog URL (preferred):** Go to `medium.com/p/import` → paste the `yarnnn.com/blog/{slug}` URL → click Import
2. **Fix title:** Remove the " | yarnnn" suffix that Medium pulls from the HTML `<title>` tag
3. **Remove logo:** Delete the YARNNN logo image that gets imported as the first image
4. **Draft saves automatically** — Medium cannot publish immediately through import. Kevin publishes manually when ready.
5. Set the **canonical URL** to the yarnnn.com/blog version (prevents duplicate content for SEO)
6. Add 3-5 relevant Medium tags
7. Kevin publishes when ready

## Medium-specific adjustments

- **Subtitle:** Add a subtitle that hooks the reader (Medium shows this prominently)
- **Pull quotes:** Add 1-2 pull quotes for visual breaks (Medium's quote formatting is good)
- **Images:** If available, add a header image. Medium posts with images get more distribution.
- **Tags:** Choose from Medium's tag system. Prioritize: "Artificial Intelligence," "AI Agents," "Productivity," "Technology," "Future of Work"
- **Author bio:** Brief bio linking to yarnnn.com

## Rules

- Every Medium post MUST set canonical URL to the blog original
- No Medium-exclusive content — everything starts on yarnnn.com/blog
- Publish within 24 hours of blog post going live
- Don't edit the Medium version independently — update the blog source, then re-sync

## Quality checklist

- [ ] Canonical URL set to yarnnn.com/blog version?
- [ ] 3-5 relevant tags selected?
- [ ] Subtitle added?
- [ ] Formatting renders correctly on Medium?
